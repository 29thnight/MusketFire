# MusketFire — 게임인재원 1Q MiniProject

스토브 데모 : https://store.onstove.com/ko/games/3679

내부 SVN 서버를 사용한 프로젝트였기 때문에 아래와 같이 작업 내용을 정리함.

[Day 1] 0510
 - 머스킷 구현에 필요한 본체, 막대, 총알 클래스 작성

[Day2] 0511
 - 이미지 회전 구현, 콜라이더 오프셋 업데이트 구현(본체가 회전하면 충돌부위 업데이트)

[Day3] 0512
 - 장전시스템 구현을 위한 상태 작성
 - 본체와 총알 상호작용을 통해 총알 장전상태 구현
 - 막대와 본체 상호작용을 통해 총알을 화약구까지 밀어넣는 로직 구현
 - 막대를 본체에 수납가능하게 작성

[Day4] 0513
 - 주말 작업본 병합 진행
 - 엔진업데이트에 따른 전반적인 클라이언트 코드 수정
 - 막대를 본체에 수납후 부착구현

[Day5] 0514
- 장전 해머 위치 플래그 별로 각도 계산 변경
- 아이템 생성시 입력한 위치에 생성할 수 있도록 매개변수 추가
- 충돌로직 변경으로 인한 클라이언트 코드 수정
- 마우스 클래스 수정

[Day6] 0515
- MusketObject, Item 추상클래스 추가
- GunPowder 아이템 추가
- Flint 아이템 추가
- Rammer 를 활용한 총알 장전부 수정
  + Rammer 수납 수정

[Day7] 0516
- 작업물 병합
- Item 클래스에 일부 함수 구현으로 인한 각 Item 비트플래그 조건 수정
- 마우스 충돌 정보로 인해 다른 오브젝트들과 상호작용 방해를 막기위한
  InteractionMouse() 함수 수정

[Day8] 0517
- 각종 버그 수정, 장전시스템 비트플래그 조건 수정
- 페이퍼 카트리지 아이템 추가
 + 페이퍼 카트리지 분리 구현

[Day9] 0518
- 작업물 병합
- 기타 아이템 작성중[Letter, Biscuit]
- 폰트파일을 불러와 장전시스템 플래그를 검사하고 / 미완료, 완료 표기 기능 추가
- 화면 아래로 떨어진 아이템, 아군진형 전장 바닥으로 위치이동기능 추가
  + 전장화면으로 나간 아이템 스케일 1/2로 축소기능 추가

[Day10] 0519
- 편지 클래스 및 편지지 클래스 작성
- 편지지의 ID에 해당하는 텍스트를 불러오기 위한 TextLoadSystem 클래스 작성중
- 각종 버그 수정(화약이 들어가는 조건이 맞지 않는 것을 수정)

[Day11] 0520
- TextLoadSystem 작성완료 및 기능 수정, LetterStruct 구조체 작성완료
- 편지 클래스 및 편지지 클래스 작성완료
- 튜토리얼 편지지 클래스를 위한 조건 추가 작성중
- 게임 기본 폰트 적용
- 전장에 드랍된 아이템의 외곽이펙트 적용을 위한 Item 클래스 함수 작성 DrawRimLight()

[Day12] 0521
- 강제 시스템 해상도 변경 및 게임 종료시 해상도 원상복귀를 위한 전역함수 작성
 + 전처리를 통한 릴리즈 모드 빌드시에 자동 적용 하도록 작성
- 대사 처리를 위한 말풍선 클래스 작성중

[Day13] 0523
- 말풍선 클래스 작성완료
- 대사 출력을 위한 머스킷 클래스 수정완료
- 말풍선 클래스 대사 ID에 해당하는 사운드 출력 연동 로직 작성
- 외곽 이펙트 변경에 따른 Item 클래스 함수 수정

[Day14] 0524
- 버그수정
